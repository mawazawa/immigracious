## Gemini 3 Pro Infographic Prompts

### INFOGRAPHIC 1: Interview Room Layout
```
Create a clean, minimal top-down floor plan diagram of a USCIS interview room. Show: officer desk at top, two applicant chairs facing desk, door on left wall, small camera icon in corner. Use teal (#0d9488) for furniture outlines, light gray for walls, white background. Label each element with clean sans-serif text. Professional infographic style, no decorative elements.
```

---

### INFOGRAPHIC 2: Process Timeline
```
Create a horizontal timeline infographic with 5 connected circular nodes showing the green card interview process: 1) "File Application" (document icon), 2) "Biometrics" (fingerprint icon), 3) "Interview Notice" (envelope icon), 4) "Interview Day" (building icon), 5) "Decision" (checkmark icon). Connect with arrows. Teal (#0d9488) accent color, white background, include estimated timeframes below each node. Modern flat design.
```

---

### INFOGRAPHIC 3: Day-Of Checklist Card
```
Create an elegant checklist card with three columns: "Bring" (document icon), "Wear" (shirt icon), "Do" (checkmark icon). Each column has 4-5 bullet items with empty checkboxes. Warm cream background (#fef3c7), teal accents, subtle drop shadow, rounded corners. Professional print-ready design, clean typography.
```

---

### INFOGRAPHIC 4: Outcome Flowchart
```
Create a vertical decision tree flowchart starting with "Interview Complete" at top, branching to 5 outcomes: "Approved" (green), "Pending Background" (blue), "RFE Issued" (yellow), "Stokes Interview" (orange), "Denied" (red). Each outcome has a brief "Next Step" underneath. Clean lines, color-coded paths, white background, professional business graphic style.
```

## Style Parameters Reference

| Parameter | Value | Purpose |
|-----------|-------|---------|
| `--ar 3:4` | Portrait | Cover, intro |
| `--ar 16:9` | Landscape | Section headers |
| `--ar 4:3` | Standard | General photos |
| `--s 250-350` | High stylize | Emotional, cinematic |
| `--s 200-250` | Medium stylize | Documentary |
| `--style raw` | Raw mode | Authentic feel |
