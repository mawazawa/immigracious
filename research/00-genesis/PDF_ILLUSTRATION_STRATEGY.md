## Part 4: Production Workflow

### Midjourney Workflow (for emotional/hero images)
```
1. Open Midjourney (midjourney.com)
2. Enter prompt with --ar and --s parameters
3. Generate 4 variations
4. Select best → Upscale (Subtle for documents, Creative for art)
5. Download high-res version
6. Place in PDF layout
```

### Nano Banana Pro Workflow (for infographics)
```
1. Open Gemini (gemini.google.com) or AI Studio
2. Enter detailed infographic prompt
3. Request "high resolution, 300 DPI, print-ready"
4. Download PNG
5. If text needs adjustment: refine in Canva
6. Place in PDF layout
```

### Moodboard Strategy (for brand consistency)
From the cheat sheet, create a Midjourney moodboard:
```
1. Go to "Personalize" → "Create Moodboard"
2. Upload 5-10 images that represent Immigracious brand:
   - Professional yet warm
   - Trustworthy blue/teal tones
   - Diverse representation
   - Government/official aesthetic
   - Hopeful outcomes
3. Copy MB Code
4. Add to all prompts: --p m[your-code]
```

## Part 7: Quick Reference - Cheat Sheet Parameters

### Most Useful for PDF Illustrations

| Parameter | Value | Use Case |
|-----------|-------|----------|
| `--ar 3:4` | Portrait | PDF pages, cover |
| `--ar 16:9` | Landscape | Section headers |
| `--ar 1:1` | Square | Icons, small graphics |
| `--s 150-200` | Medium stylize | Professional, clean |
| `--s 300+` | High stylize | Artistic, emotional |
| `--style raw` | Raw mode | Natural, documentary |
| `--sref [code]` | Style reference | Brand consistency |
| `--no text` | Exclude text | Clean images for overlay |
